FAQ

What is Curve.fi?

Curve.fi is a set of smart contracts which create on-chain liquidity for stablecoins in a very efficient, autonomous way. That means smaller price slippage for traders and higher profits for liquidity providers.

How to trade?

Just go to the exchange page and do it! For now, you can exchange between DAI and USDC, there will be more currencies in nearest future.

How to use Curve.fi for profit?

Go to the deposit page and add some cDAI / cUSDC. For that, supply some DAI / USDC, deposit on Compound Protocol. Then, you'll be able to deposit DAI and USDC which are getting interest on Compound. So you can earn trading fees and compounded interest at the same time!

What's "Add all coins in a balanced proportion"?

When you are typing, how many coins to add, you need to add them in the same proportion as the current pool. If you don't get that option checked, you will automatically exchange your coins to be in balance, which is recommended if you are adding a significantly smaller amount than the current liquidity.

What's "Use maximum amount of coins available"?

This means using all USDC and DAI in your wallet. This way is recommended only if you have much less coins than currently in liquidity pool.

How to withdraw liquidity I provided

Go to the withdraw page. If you want to withdraw some percentage of your liquidity (the preferred way), type that percentage in the top field. You can, however, withdraw in a form of individual coins (USDC, DAI), having the exchange happening for you, if you type amounts in lower fields. You'll get charge with the exchange fee in the latter case.
